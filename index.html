Change Log
Done
Version 1.2a

This version contains a whole lot of backend changes and the ability to send a one-time message from server to client about important application information.

Version 1.2

This version adds user interface to loading and saving 5x5s. The app can now load and save 5x5s for any teacher!

Version 1.1c

This version fixes a relatively long-standing bug with the connection state banner where it would fire at inappropriate times or not at all. It now fires correctly and prevents the user from submitting 5x5s when not online.

Version 1.1b

This version is a minor bugfix release

Version 1.1a

This version fixed bugs with the update banner text (it was poorly worded). It also added an "about" page and cemented the version numbering system.

Version 1.1

This update was the first major version bump! It did the following:

  • Renamed review categories according to new specifications
  • Renamed one list item as per administrative request
  • Added the Rigor/Relevance Framework picker and example image
  • Changed behind-the-scenes code to handle user-facing changes

Version 1.0

Woo! Finally out of beta! This is the initial release of the application. It has:

  • A complete suite of tools necessary to conduct 5x5s!
  • Secure user authentication!
  • Big green buttons!
  • Offline functionality!*
  • Decent-looking graphics assets!
Here's to more exciting developments for my first real development project!
* Restrictions apply. Limited to only 5x5s for the teacher "Admin Admin".
